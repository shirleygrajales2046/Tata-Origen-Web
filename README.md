# Tata Origen Web

Aplicación web en desarrollo para la gestión de mi emprendimiento Tata Origen.

## Tecnologías
- React
- TypeScript
- Vite
- CSS

## Descripción
Este proyecto tiene como objetivo organizar y gestionar productos, pedidos y procesos relacionados con mi emprendimiento.

Actualmente se encuentra en desarrollo y se planea integrar un backend para manejo de datos y lógica de negocio.

## Objetivo
Aplicar conocimientos en desarrollo web full stack, especialmente enfocados en backend y manejo de bases de datos.

## Estado del proyecto
En desarrollo 🚧

## Aprendizajes
- Estructuración de proyectos frontend
- Manejo de componentes en React
- Organización de código escalable
